Setting of dotvm
====

dotV setup vim with vundle

How to distribute dotV
---

Install:
----
    git clone git://github.com/gogonkt/dotV.git ~/.vim
    ln -s ~/.vim/vimrc ~/.vimrc
then launch vim, and run:
    :BundleInstall
done!

Add plugins and vundle usage:
----
https://github.com/gmarik/vundle
